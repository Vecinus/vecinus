# 5-S2-ai-declaration.md

**AI Use Declaration**

Artificial intelligence tools were used during the development of this project as a support resource. Specifically, the following models and tools were utilized:

- **Gemini 3.1:** Used for all AI-assisted tasks, including code generation, debugging, improving documentation grammar and clarity, brainstorming ideas, and providing conceptual explanations.
- **Claude:** Used for code generation, debugging and code styles.

All AI-generated suggestions were thoroughly reviewed and adapted by the team. All core design decisions, architectural implementation, and final validation of the application were carried out by the authors. The team remains fully responsible for the functionality, originality, and integrity of the final project.
