# IMPORTANTE!!!!

Cuando pase a producción debemos añadir un .env con la siguiente clave para que funcione bien la aplicación.
Mientras tanto, hay que añadir la ip publica de tu pc.
EXPO_PUBLIC_BACKEND_URL=http://X.X.X.X:8000
Debéis quitar el firewall porque sino no deja compartir el puerto (experiencia propia)
EXPO_PUBLIC_BACKEND_URL=https://tu-backend-en-produccion.com
